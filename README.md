# CodeBuzz-Relay
CodeBuzz Relay is a DIY handheld pager powered by the XIAO ESP32-S3 and LoRa technology. Designed for off-grid communication, it features a 2.8" horizontal TFT display and a custom-engineered 5x7 keyboard matrix with a dedicated D-pad. No Wi-Fi, no cellular—just pure, long-range mesh messaging.
