# Features
This design features a Seed Studio XIAO ESP32-S3 & Wio-SX1262 Kit. It Features an 2.8-Inch Colour tft display, a
joystick for navigation, and it's battery-powered, 5x7 matrix. More info will be added later,

# Repository
This repository has all the project files that you will need to understand, refine, and build it
In the folder named PCB_FILES, it contains KiCad files, Gerbers, and drill files, and in the folder named enclosure, it contains step files and a file on Onshape 
in the file named BOM.URL is a link to the Excel sheet containing the Bill of Materials
And in the file named Fallout zine, it's a poster for the project
# Enclosure
The enclosure is designed to make the device look like a finished product. It can accommodate the Mesh-IT PCB and a 3000 Mah Battery (50 x 50 x 10mm). I recommend using Clear PETG to highlight the internals, which soulfully makes the project beautiful
# Assembly instructions
1. Solder all components on PCB
2. Mount the display on the top plate and align it with the display cutout
3. Screw the display into the top plate using screws and nuts
4. Place the PCB inside the enclosure and check that the keyboard, joystick, display, and ports line up properly
5. Glue/epoxy the charging modules onto the small platform in the top plane
6. Insert the heat inserts into the backplate
7. Place the battery inside the enclosure
8. Align the top plate, the PCB, and the backplate
9. screw everything together carefully.
10. More detailed instructions will be added after I physically build and test the device
